# Winter-Project-2024 – To-Do List Manager

This project is a small **To-Do List** application written in Java.  
It runs in the console and lets you manage simple tasks.

## Features

- Add a task (ID, title, due date).
- View all tasks.
- View only pending tasks.
- Mark a task as done.
- Save and load tasks from `tasks.txt`.

1. Install Java (JDK) on your system.
